# Введение в B4

**B4 (Bye Bye Big Bro)** — мощный инструмент для обхода систем глубокой инспекции пакетов (DPI) на базе Linux netfilter.

## Что такое B4?

B4 перехватывает и модифицирует сетевые пакеты в реальном времени, применяя различные техники для обхода DPI-систем:

- **Фрагментация TCP/IP** — разбивает пакеты для обхода детекции
- **Фейковые SNI-пакеты** — отправляет ложные данные для запутывания DPI
- **Поддержка UDP/QUIC** — работает с современными протоколами
- **Высокая производительность** — многопоточная обработка пакетов
- **Гибкая настройка** — множественные конфигурационные сеты

## Возможности

### Веб-интерфейс

- Метрики и статистика в реальном времени
- Потоковый просмотр логов с фильтрацией
- История подключений с информацией о протоколах
- Управление доменами и IP-адресами
- Визуальная настройка конфигурации

### Конфигурационные сеты

Создавайте различные стратегии обхода для разных сценариев:

- Отдельные настройки TCP/UDP для каждого сета
- Индивидуальные стратегии фрагментации
- Настройка фейковых пакетов под задачу
- Приоритизация сетов

### Интеграция GeoSite/GeoIP

- Поддержка баз данных v2ray/xray
- Категории доменов (youtube, netflix, и т.д.)
- Автоматическое обновление баз
- Комбинирование ручных и автоматических списков

## Быстрый старт

Установка одной командой:

```bash
wget -O ~/b4install.sh https://raw.githubusercontent.com/DanielLavrushin/b4/main/install.sh && \
chmod +x ~/b4install.sh && \
~/b4install.sh
```

После установки:

```bash
# Запуск сервиса
sudo systemctl start b4
sudo systemctl enable b4

# Открыть веб-интерфейс
http://ваш-ip:7000
```
