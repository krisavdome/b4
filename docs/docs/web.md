# Веб-интерфейс

По умолчанию доступен на `http://IP-роутера:7000`

## Быстрый старт

```bash
# Запустите сервис
systemctl start b4         # systemd
/etc/init.d/b4 start       # OpenWRT
/etc/init.d/S99b4 start       # MerlinWRT

# Откройте браузер
http://192.168.1.1:7000
```

Всё. Интерфейс работает без авторизации из локальной сети.
